# spring-deepstart-app

Spring Boot を使った学習用アプリケーションです。  
フォーム入力 → バリデーション → 確認画面 → 完了画面 という一連の流れを実装しています。

---

## 📌 主な機能

- 名前とメールアドレスの入力フォーム
- バリデーション機能（未入力・メール形式チェック）
- 確認画面での入力内容の表示
- 登録完了画面の表示

---

## 🚀 使用技術

- Java 17
- Spring Boot 3.x
- Thymeleaf
- Spring Validation（jakarta.validation）
- Maven
